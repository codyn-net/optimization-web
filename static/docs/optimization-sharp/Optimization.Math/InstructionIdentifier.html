InstructionIdentifier Class

Identifier instruction.

public class InstructionIdentifier : Instruction

⊟ Remarks

An identifier instruction is an instruction that will be resolved while executing the instruction set. The identifier is looked up in the context supplied to the expression evaluation.

public override void Execute (Stack<double> stack, Dictionary<string, object> context)

⊟ Parameters

stack
The execution stack.
context
The execution context.

⊟ Remarks

Lookup the identifier in the context and if found execute the value. If the value is another Optimization.Math.Expression it will be evaluated in turn (with the same context and stack). If the identifier could not be found, 0 is pushed on the execution stack.
