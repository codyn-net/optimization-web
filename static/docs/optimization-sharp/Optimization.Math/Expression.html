Expression Class

Mathematical expression representation.

public class Expression

⊟ Remarks

This class represents a mathematical expression. It can parse and evaluate most common mathematical operators and a limited set of mathematical functions (such as sin, cos, abs, min, max, etc). It also has support for variables which will be substituted on evaluation Expression.Evaluate
