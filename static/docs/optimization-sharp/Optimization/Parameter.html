Parameter Class

Parameter class.

public class Parameter : ICloneable

⊟ Remarks

Optimization parameter class. This class stores a single parameter used for optimization. Each solution in the optimization as a certain set of parameters which make up the solution. A parameter consists of a name, a value and a boundary by which the parameter value is bound.
